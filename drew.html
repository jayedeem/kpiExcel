Here are some of the things I learned.

Chapter 1 - Getting Started with Excel

  • Using the key commands Cntrl + Home will bring you to the first cell

    • Chapter 2 - Entering Data

  • Numbers and formulas are right-aligned, while text is left-aligned

    • Chapter 3 - Creating Formulas and Functions

  • Formulas start with ="Function"(paramaters)
  • An absolute cell reference is a cell reference in a spreadsheet application that remains constant even if the shape or size of the spreadsheet is changed,
    or the reference is copied or moved to another cell or sheet. Absolute cell references are important when referring to constant values in a spreadsheet.
  • Excel favors calculations from above to bellow cells

    • Chapter 4 - Formatting

  • You can change the numerical value of a number by highlighting the cell and right clicking and choosing
    Format or using the key command cntl + 1
  • Autofill can drag left, right, down, or up

    • Chapter 5 - Adjusting Worksheet Layout and Data

  • You can drag selected cells by highlighting the cells you want moved and going to the
    edge when the cursor turns in a cross with arrows at the end. You then can drag to any open cells.
  • You can also copy selected cells and move them freely to an open cell by doing the same steps above, but
    you are holding down the cntrl button and dragging at the same time.

    • Chapter 6 - Printing

  • Print Preview key commands is cntrl+F2
  • You can print out certain cells by using Print area which is found under the Page Layout.
    Highlight the cells you want printed out and click Print area -> Set Print Area. Alternatively, you can
    hightlight the cells and key command cntrl + p and under settings choose "Print Selection."
  • Page Break preview can determine what parts of the workbook will cut off when it is printing. To see this
    where where this will cut off head to the bottom right corner next to the zoom function, you will see 3 different
    icons. Pick the last one to see the page break. Pulling from the right edges of
    book you can manipulate how much of the book you want to print.
